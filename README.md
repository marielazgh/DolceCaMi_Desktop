# DolceCaMi_Desktop
# Scopul aplicației: 
Scopul aplicației Dolce Cami este de a oferi utilizatorilor o platformă digitală modernă și intuitivă pentru explorarea meniului restaurantului, consultarea ofertelor și plasarea comenzilor într-un mod rapid și eficient. Aplicația urmărește să îmbunătățească experiența clienților prin prezentarea preparatelor într-un format atractiv, facilitarea accesului la informații despre livrare și simplificarea interacțiunii dintre restaurant și utilizator. Prin designul său inspirat din atmosfera italiană și funcționalitățile ușor de utilizat, Dolce Cami are ca scop să transforme procesul de alegere și comandare a mâncării într-o experiență plăcută, accesibilă și profesionistă.

# Structura proiectului
Proiectul aplicației Dolce Cami este organizat pe mai multe componente principale, fiecare corespunzând unei funcționalități sau unei etape a aplicației. Structura poate fi descrisă astfel:

- Pagina Home:
  Introducere în restaurant, prezentarea conceptului și a atmosferei italiene
  Elemente vizuale: imagini, logo, scurt mesaj de bun venit
- Pagina Oferte:
  Afișarea promoțiilor și ofertelor speciale
  Carduri interactive cu descriere, imagini și perioadă de valabilitate
- Pagina Meniu:
  Catalog cu preparatele restaurantului împărțite pe categorii (paste, pizza, deserturi, băuturi)
  Posibilitatea de a vedea detalii despre fiecare preparat și de a adăuga în coș
- Pagina Livrare:
  Informații despre metodele de livrare și costuri
  Formular de completare a datelor pentru plasarea comenzilor
- Pagina Login:
  Autentificare utilizator pentru a accesa funcțiile de comandă și istoricul comenzilor
- Structura tehnică / prototip Figma:
  Wireframe-uri și prototip interactiv care reflectă fluxul aplicației
  Legături între pagini și butoane interactive pentru testarea navigării
